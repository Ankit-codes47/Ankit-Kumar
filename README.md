# 🚀 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website designed to showcase my journey as a second-year engineering student, along with my skills, projects, academic progress, certificates, and achievements.

The website features a premium dark-themed UI with glassmorphism, bento-grid layouts, smooth animations, interactive hover effects, and a fully responsive design.

## ✨ Highlights

- Modern Dark UI
- Responsive Bento Grid Layout
- Interactive Profile Section
- About & College Information
- Project Showcase
- Academic Results Timeline
- Skills & Certificates
- Magnetic Button Effects
- 3D Project Card Tilt
- Mouse-Tracking Glow Effects
- Text Scramble Animations
- Responsive Contact Section

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome

## 🎯 Purpose

This portfolio represents my growth as an engineering student and developer. I will continue updating it as I learn new technologies, build new projects, earn certifications, and progress through my engineering journey.

> **Learning. Building. Improving — one project at a time.**
